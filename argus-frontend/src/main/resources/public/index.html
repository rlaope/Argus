Argus Guide

What is Argus?

Argus is a real-time JVM observability platform powered by JFR (Java Flight Recorder). It monitors virtual threads, GC, CPU, memory, allocation, lock contention, and method profiling.

Metrics Explained

Total Events
The cumulative count of all JFR virtual thread events received since the dashboard was opened.
Started
Number of jdk.VirtualThreadStart events.
Ended
Number of jdk.VirtualThreadEnd events.
Active
Estimated number of currently running virtual threads (Started - Ended).
Pinned
Number of jdk.VirtualThreadPinned events. Critical metric! Pinning occurs when a virtual thread cannot be unmounted from its carrier thread, typically due to:
  • synchronized blocks or methods
  • Native method calls (JNI)
High pinning degrades virtual thread scalability.

Thread View

Visual representation of active threads as cards. Each card shows thread name, duration, and carrier thread ID.

Pinned Alerts

When a thread is pinned, it appears in the alert section with its stack trace. Use this to identify and fix blocking code.
