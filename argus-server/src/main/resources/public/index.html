Argus Guide

What is Argus?

Argus is a real-time JFR (Java Flight Recorder) event monitor for Java Virtual Threads (Project Loom).

Metrics Explained

Total Events
The cumulative count of all JFR virtual thread events received since the dashboard was opened.
Started
Number of jdk.VirtualThreadStart events. Each event represents a new virtual thread beginning execution.
Ended
Number of jdk.VirtualThreadEnd events. Each event represents a virtual thread completing its execution.
Active
Estimated number of currently running virtual threads (Started - Ended). Note: This is an approximation based on events received.
Pinned
Number of jdk.VirtualThreadPinned events. Critical metric! Pinning occurs when a virtual thread cannot be unmounted from its carrier thread, typically due to:
  • synchronized blocks or methods
  • Native method calls (JNI)
High pinning degrades virtual thread scalability.

Thread View

Visual representation of active threads as cards. Each card shows:

  • Thread Name - The virtual thread identifier
  • Duration - How long the thread has been running
  • Carrier - The platform thread ID carrying this virtual thread

Pinned Alerts

When a thread is pinned, it appears in the red alert section with its stack trace. Use this to identify and fix blocking code.
